# Интеграция логотипов CampusFlow

## Размещение файлов логотипов

Разместите ваши SVG файлы в следующих местах:

### 1. Минималистичный логотип CampusFlow
**Файл:** `schedule-hub-uni-main/public/logos/campusflow-minimal.svg`
**Использование:** Иконка в Header, favicon, круглые аватары
**Рекомендуемые размеры:** 32x32px, квадратный формат
**Где используется:**
- Header (в круглом контейнере)
- Главная страница (большая иконка)
- Favicon браузера

### 2. Горизонтальный логотип CampusFlow
**Файл:** `schedule-hub-uni-main/public/logos/campusflow-horizontal.svg`
**Использование:** Полный логотип с названием
**Рекомендуемые размеры:** произвольная ширина x 48px высота
**Где может использоваться:**
- Футер (если добавите)
- Документы
- Презентации

### 3. Логотип КГУ
**Файл:** `schedule-hub-uni-main/public/logos/ksu-logo.svg`
**Использование:** Официальный логотип университета
**Рекомендуемые размеры:** 48x48px
**Где используется:**
- Header (рядом с CampusFlow)
- Главная страница (партнерский логотип)

## Компонент Logo

Создан универсальный компонент `<Logo>` для работы с логотипами:

```tsx
// Минималистичный логотип
<Logo variant="minimal" size="lg" />

// Горизонтальный логотип  
<Logo variant="horizontal" size="md" />

// Логотип КГУ
<Logo variant="ksu" size="md" />
```

### Доступные размеры:
- `sm` - маленький (24px)
- `md` - средний (32px) 
- `lg` - большой (48px)
- `xl` - очень большой (64px)

## Замена логотипов

1. **Сохраните ваши SVG файлы** в указанных путях
2. **Убедитесь в корректности путей** - файлы должны быть доступны по URL
3. **Проверьте размеры** - логотипы должны хорошо смотреться в разных размерах
4. **Обновите страницу** - изменения применятся автоматически

## Рекомендации по SVG

- **Используйте currentColor** для элементов, которые должны наследовать цвет
- **Оптимизируйте размер** SVG файлов для быстрой загрузки  
- **Тестируйте на разных фонах** - особенно для Header с градиентом
- **Убедитесь в читаемости** на мобильных устройствах

## Брендинг

Проект теперь использует двойной брендинг:
- **CampusFlow** - как название системы/продукта
- **КГУ им. К.Э. Циолковского** - как партнер/заказчик

Это подчеркивает, что CampusFlow - это современная система, созданная для КГУ.
